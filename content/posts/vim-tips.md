## 移动光标

| 按键 | 操作 |
|------|------|
| `h` `j` `k` `l` | 左、下、上、右 |
| `w` `b` | 下一个/上一个单词 |
| `0` `$` | 行首/行尾 |
| `gg` `G` | 文件开头/结尾 |
| `Ctrl+d` `Ctrl+u` | 向下/向上翻半页 |

## 编辑操作

```bash
# 删除
dd    # 删除当前行
dw    # 删除一个单词
d$)   # 删除到行尾

# 复制粘贴
yy    # 复制当前行
p     # 粘贴到光标后
P     # 粘贴到光标前

# 撤销重做
u     # 撤销
Ctrl+r # 重做
```

## 高效组合

Vim 的强大在于命令可以组合：

- `ci"` — 清空双引号内内容并进入插入模式
- `dt=` — 删除从光标到等号之间的内容
- `>G` — 从当前行缩进到文件末尾

> 记住：Vim 不是编辑器，是一种思维方式。
